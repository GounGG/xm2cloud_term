# xm2cloud_term
#### 简单介绍
> [xm2cloud_term](https://github.com/xmdevops/xm2cloud_term)基于Django+Channels+Xterm+IOLoop实现,目前作为[xm2cloud_cmp]()平台基础组件.

***

#### 开发环境
> SY_ENV: MacOS 10.12.6 \
> PY_ENV: Python2.7.10

#### 安装部署
> 从项目作为独立Django App载入您Project即可,有好的建议或任何疑问欢迎提ISSUES.
